# Voyager
A partial backend to a course system. Hacky at best, not currently maintained

## Usage

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'voyager'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install voyager
```

### Finally

Run - rails voyager:install:migrations
Check - routes.rb
